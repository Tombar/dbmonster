Overview

DataGenerator is a very simple mechanism to generate.... yes, data! In context of DBMonster we bind one data generator to each column and this generator becames a values factory for that column. There are several default generators, which will be discussed later. The important thing is that you may write your own generator and add it easly to DBMonster.

Default generators

  • String generator
  • Boolean generator
  • Timestamp/date/time generator
  • Number generator
  • Null generator
  • Foreign key generator
  • Dictionary generator
